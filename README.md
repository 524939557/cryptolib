# cryptolib

### Data Encoding (dec)
* [BETA] - HEX
* [WIP] - ASCII
* [   ] - BINARY
* [   ] - DECIMAL
* [   ] - BASE64

### Cryptographic hash functions (chf)
* [   ] - MD5
* [   ] - SHA1
* [   ] - SHA256
* [   ] - SHA512
* [   ] - HMAC
* [   ] - BCRYPT
* [   ] - PBKDF2
* [   ] - SCRYPT

### Symmetric-key cryptography (skc)
* [   ] - AES
* [   ] - Blowfish
* [   ] - Serpent
* [   ] - Twofish

### Public-key cryptography (pkc)
* [   ] - RSA

* [   ] - Curve1174
* [   ] - MDCurve201601
